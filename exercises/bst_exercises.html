int count_last_level() const

      

      
Counts the number of nodes in the last (deepest) level of the tree.

      
      

        
Approach

        

          1. Call compute_heights() and compute_depths().

          2. The depth of the last level equals the height of the root.

          3. Perform any traversal and count all nodes whose depth equals root->height.
