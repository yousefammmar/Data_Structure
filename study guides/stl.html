What Is The STL?

The STL (Standard Template Library) is a built-in C++ library offering a wide range of container classes implemented using the data structures from this course. Key containers include:

stack
Linked-list or array based
queue
FIFO ordering
vector
Dynamic array
list
Doubly-linked list
set
Balanced BST, no duplicates
map
Balanced BST, key→value
unordered_set
Hash table
priority_queue
Heap-based
What you need to know for this course: Basic operations, running times, range-based loops, and when to use which container — for vector, list, stack, queue, set, and map.

The set Container


    

      

        
🌳

        
set — Balanced BST, No Duplicates

      

      
Implemented as a balanced BST. All operations run in O(log n). Elements are always iterated in ascending order.

      

        
FunctionDescriptionTime

        
          
insert(val)Insert val (if not present)O(log n)

          
erase(val)Remove valO(log n)

          
count(val)1 if present, 0 otherwiseO(log n)

          
clear()Remove all elementsO(n)

          
empty()Check if emptyO(1)

          
size()Number of elementsO(1)

The map Container


    

      

        
🗺️

        
map — Key → Value Pairs

      

      
Implemented as a balanced BST where each node stores a key-value pair. Keys are unique; values can repeat. Iterating gives keys in ascending order.

      

        
FunctionDescriptionTime

        
          
mymap[key] = valInsert or update pairO(log n)

          
mymap[key]Return value for key (inserts if missing!)O(log n)

          
erase(key)Remove keyO(log n)

          
count(key)1 if key exists, 0 otherwiseO(log n)

          
clear()Remove all pairsO(n)

          
empty()Check if emptyO(1)

          
size()Number of key-value pairsO(1)
